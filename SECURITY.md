# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Perk Protocol, please report it responsibly.

**Email:** nebula7458@proton.me

**Do not** open a public GitHub issue for security vulnerabilities.

## Scope

- Perk Protocol smart contracts (Anchor/Rust)
- TypeScript SDK (`perk-protocol`)
- Frontend application (perk.fund)
- Oracle system (PerkOracle)

## What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response

We will acknowledge your report within 48 hours and provide an initial assessment within 7 days.
